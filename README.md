# 玛吉斯APP
###### 注：项目采用分支开发流程，具体如何使用参考技能栈：BS038

### 项目环境
###### 项目为 uni-app 项目，建议安装 HBuilderX 编辑器，以便调试运行

### 项目运行
###### 浏览器调试：编辑器菜单栏【运行】-【运行到浏览器】-【Chrome】
###### 真机调试（安卓）：编辑器菜单栏【运行】-【运行到手机或模拟器】-具体手机
###### 注：安卓调试需要打开手机【开发者模式】中的【允许USB调试】开关

### 项目发行
###### 编辑器菜单栏【发行】- 原生APP打包
